woa7's NETWORK BOOT

Network boot Linux, BSD, DOS and tools like hdt, memtest, ...

Legend:
  • (s) - signed and signature is verified before boot
  • (m) - booting from an official mirror
Requirements:
  • x86 (32 or 64 bit) PC with network connection
Optional but recommended:
  • network configuration from DHCP server
  • PXE enabled network card

FAQ

  • How to start?
    Look into ipxe/ dir for how to boot your favorite system.
  • What is the shortest way to chainload this menu from another ipxe/gpxe (for example from qemu's internal pxe)?
    chain http://woa7.github.io
    No need to specify filename, ipxe/gpxe user-agent is autodetected and proper content will be sent.
  • What is the shortest way to call other scripts, which are not present in menu?
    Just type "chain scriptname". For example "chain lspci". No need to type ".ipxe" extension, it will be added automatically.
  • Is it opensource?
    YES! It's released under GPL v2. You can download sources from my git repository:
    git clone https://github.com/woa7/woa7.github.io

    based on the source of http://boot.salstar.sk / http://git.salstar.sk/pxeboot.git
  • Do you have more questions/suggestions?
    Contact me via https://github.com/woa7/woa7.github.io/issues/new/choose
